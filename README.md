
# MusicGen Test 🎵

A quick experiment with Facebook's MusicGen model to generate music from text prompts.

## What this does

Feeds a text description to MusicGen and outputs a .wav audio file.

## How to run

pip install transformers torch scipy

python main.py

## Example prompt

"dark trap with hard 808 and drums"

## Notes

This is just me learning how MusicGen works.
Built while following a fine-tuning tutorial.
Next step → fine-tune it on my own music.

## Requirements

- Python 3.8+
- ~1GB disk space (model download)
- GPU optional but faster
